Eliminating distractions

January 31, 2019

A week into the announcement, I realised that I have already created some debt for myself. I am calling it time debt as it is taking much of my time.

Photo by Christian Wiediger on Unsplash

I created few public channels for open discussions about the project only to realize that they would start to take more time than letting me focus on the actual project.

In retrospect, I have removed all those channels now that included a Twitter handle and a Facebook group. However, I am going to keep the long-standing Facebook Page – the only place to keep up with project updates.

There was also an intent of keeping everything organised in this project. I think I went a bit extreme there too by dividing the project into parts on GitHub organisation. Therefore retracting the overdone part and keeping main project together in one place makes sense. The content still live in their own repository and remain independent of the application.

Hoping that at least, this won’t derail me from the main focus and I continue to work on adding more content for the launch. I am available on Twitter (@jabranr) for any discussions in this regard.
